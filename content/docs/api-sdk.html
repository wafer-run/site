Guest SDK

Three SDKs are available for writing WASM blocks: Rust (wafer-sdk), Go (wafer-sdk-go), and TypeScript (@wafer-run/sdk). All three generate the same four WASM exports (malloc, info, handle, lifecycle) that the runtime calls. Use the CLI to scaffold projects with the correct setup for each language.

Rust SDK (wafer-sdk)

The Rust SDK uses a proc macro to generate the block boilerplate from simple attribute args. Add to Cargo.toml:

[lib]
crate-type = ["cdylib"]

[dependencies]
wafer-sdk = "0.1"
serde = { version = "1", features = ["derive"] }
serde_json = "1"

[profile.release]
opt-level = "s"
lto = true
strip = true

#[wafer_block] proc macro

Annotate your impl block to generate the WaferBlock trait impl and all four WASM exports. The handle method is required; lifecycle is optional and defaults to Ok(()):

use wafer_sdk::*;

struct MyBlock;

#[wafer_block(
    name = "myorg/my-block",
    version = "1.0.0",
    interface = "handler@v1",
    summary = "A demo block",
    instance_mode = "singleton",   // optional, default: singleton
    requires = []                  // optional, default: []
)]
impl MyBlock {
    fn handle(msg: Message) -> BlockResult {
        json_respond(&msg, &serde_json::json!({"ok": true}))
    }

    fn lifecycle(event: LifecycleEvent) -> Result<(), WaferError> {
        Ok(())
    }
}

Go SDK (wafer-sdk-go)

Install with:

go mod init myorg-my-block
go get github.com/wafer-run/wafer-sdk-go

Block interface

Implement the Block interface and call wafer.Export() from main():

package main

import wafer "github.com/wafer-run/wafer-sdk-go"

type MyBlock struct{}

func (b *MyBlock) Info() wafer.BlockInfo {
    return wafer.BlockInfo{
        Name:      "myorg/my-block",
        Version:   "1.0.0",
        Interface: "handler@v1",
        Summary:   "A demo block",
    }
}

func (b *MyBlock) Handle(ctx *wafer.Context, msg *wafer.Message) *wafer.Result {
    return wafer.JSONRespond(msg, map[string]interface{}{"ok": true})
}

func (b *MyBlock) Lifecycle(ctx *wafer.Context, event wafer.LifecycleEvent) error {
    return nil
}

func main() {
    wafer.Export(&MyBlock{})
}

TypeScript SDK (@wafer-run/sdk)

Install with:

npm install @wafer-run/sdk

SDK types are available now. WASM compilation is coming soon — TypeScript blocks can be used for type checking and local testing today.

block() factory

Use the block() factory function to define a block:

import { block, Message, BlockResult } from "@wafer-run/sdk";

export default block({
    name: "myorg/my-block",
    version: "1.0.0",
    interface: "handler@v1",
    summary: "A demo block",

    handle(msg: Message): BlockResult {
        return msg.jsonRespond({ ok: true });
    },

    lifecycle(event: LifecycleEvent): void {
        // optional
    },
});

Service Clients

All three SDKs provide typed clients for platform services. These wrap call_block() calls to the corresponding service blocks. See Clients API for the full reference.
